# ThemeWinter

## Description

A cool winter theme for Matomo with ice blues, steel navy backgrounds, and silver frost accents. Evokes crisp winter days and frost.

This theme transforms your Matomo dashboard with a clean, cool color palette featuring:

- Icy white and pale blue backgrounds with cool dark blue-gray text
- Steel navy header with frost blue accents
- Custom graph color series in frost blues, pine teals, twilight purples, and holly reds
- Visitor map styled with icy blue gradients and golden highlights

![ThemeWinter Dashboard](screenshots/Winter_Dashboard.png)

## Installation

Install it via Matomo Marketplace or manually:

1. Download the plugin
2. Extract the files into the `plugins/ThemeWinter` directory
3. Activate the theme in **Administration > Platform > Themes**
